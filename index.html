1- Formateo e Instalación de Sistema Operativo

Para este servicio deberás de guardar todos tus documentos importantes en un dispositivo de almacenamiento externo (USB) con anticipación, ya que se eliminarán todos tus datos. Formateamos tu computadora y le instalamos un sistema operativo (Windows 7, 8.1 o 10) para su funcionamiento. Incluye Sistema Operativo activado.

2- Formateo e Instalación de Sistema Operativo + Programas Complementarios

Para este servicio deberás de guardar todos tus documentos importantes en un dispositivo de almacenamiento externo (USB) con anticipación, ya que se eliminarán todos tus datos. Formateamos tu computadora y le instalamos un sistema operativo (Windows 7, 8.1 o 10) para su funcionamiento. Incluye Sistema Operativo activado.
Se le instalará a tu computadora 5 programas importantes para que esté lista a usar desde el momento que la recibas.

  • Office 2013(Activado)
  • Antivirus(AVG)
  • Chrome(Navegador)
  • Winrar(Descomprimidor/Comprimidor de archivos)
  • Adobe Reader(Lector de archivos PDF)

Q150.00

Limpiamos de manera profunda todos los componentes internos de tu CPU con productos de calidad, eliminando todo el polvo para evitar futuros problemas y dejándola con un mejor aspecto. (Incluye cambio de Pasta térmica)

Q100.00

4- Limpieza externa e interna (Laptop)
